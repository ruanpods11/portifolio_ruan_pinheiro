# ⚖️ Hocks Advocacia - Landing Page

Esta é uma landing page institucional para um escritório de advocacia, focada em um design **premium e responsivo**. 

## 🚀 Evolução do Projeto
Este projeto marca um ponto importante nos meus estudos (menos de 40 horas totais), onde refatorei um código inicial focado em posicionamento absoluto para uma estrutura moderna utilizando **CSS Grid e Flexbox**.

## 🛠️ Tecnologias Utilizadas
*   **HTML5 Semântico**: Uso de tags como `<header>`, `<section>` e `<footer>`.
*   **CSS3 Moderno**: 
    *   Variáveis (`:root`) para paleta de cores.
    *   Efeito Glassmorphism no menu fixo.
    *   Layouts com Grid e Flexbox.
*   **JavaScript Vanilla**: Lógica de menu hambúrguer e melhoria de UX no scroll.

## 📱 Responsividade
O site foi otimizado para:
*   Desktops (Telas grandes)
*   Tablets
*   Smartphones (Menu mobile interativo)
